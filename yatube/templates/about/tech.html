Проект реализован с помощью следующих технологий:

  • Python 3.9.5
  • Django 2.2.16
  • Bootstrap 5

Краткое описание функционала:

Сервис позволяет пользователям создавать свою страницу, подписываться на страницы других пользователей и формировать новостную ленту из подписок.

Кроме того, пользователи могут просматривать страницы всех других авторов и оставлять комментарии к записям. При этом посты можно отнести к разным группам, что позволяет пользователям находить авторов по интересам.

Авторизованные пользователи также могут полностью редактировать все свои сообщения и комментарии.
